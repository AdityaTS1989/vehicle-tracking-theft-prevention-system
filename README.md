# 🚗 Vehicle Tracking & Theft Prevention System

## 📌 Project Overview

This project simulates a Vehicle Tracking and Theft Prevention System using Python.

The system monitors vehicle movement, GPS location, speed, ignition status, and geofence boundaries to identify suspicious activity and generate alerts.

---

## 🚀 Features

* GPS Route Tracking
* Geofence Monitoring
* Vehicle Speed Monitoring
* Overspeed Detection
* Ignition Status Monitoring
* Theft Detection Logic
* Alert Generation System
* Safety Analytics Dashboard
* Vehicle Route Visualization
* CSV Data Export

---

## 🛠 Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Jupyter Notebook
